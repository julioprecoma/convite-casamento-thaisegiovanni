# Marco 0 - Home limpa

Versão reconstruída do zero.

## O que contém
- `index.html` com envelope inicial.
- Ao tocar: envelope some e aparece a Home.
- Igreja em aquarela como imagem principal de fundo.
- Menu alinhado no topo esquerdo.
- Botões menores e menos dominantes.
- Páginas internas apenas como placeholders.

## Como subir no GitHub
1. Descompacte o ZIP.
2. Envie o conteúdo da pasta para a raiz do repositório.
3. Confirme que `index.html` está na raiz, não dentro de outra pasta.
4. Abra com `?marco0=1` no final da URL para evitar cache.

Exemplo:
https://julioprecoma.github.io/convite-casamento-thaisegiovanni/?marco0=1

## Ajustes rápidos no index.html
Procure por estas variáveis no começo do CSS:

- `--menu-w`: largura do menu.
- `--menu-top`: distância do topo.
- `--menu-left`: distância da esquerda.
