# Convite Thaís & Giovanni — Home v1

Primeira versão funcional da página de abertura.

## Arquivos principais
- `index.html`: abertura com envelope, transição e menu.
- `assets/css/style.css`: estilo da Home.
- `assets/js/envelope.js`: controle da animação.
- páginas internas: placeholders para manter os links do menu funcionando.

## Teste local
Abra `index.html` no navegador.

## GitHub Pages
Suba todos os arquivos para o repositório e publique a branch principal no GitHub Pages.
