# Thaís & Giovanni — Home v4

Esta versão foi ajustada para evitar o problema de CSS não carregado no GitHub Pages.

## O que mudou

- O CSS e o JavaScript da Home foram incorporados diretamente dentro do `index.html`.
- Isso evita que a página apareça quebrada com ícones pretos gigantes e textos sem formatação.
- As imagens continuam na pasta `assets/img/`.
- As páginas internas continuam como placeholders.

## Como substituir no GitHub

1. Descompacte este ZIP.
2. Entre no repositório no GitHub.
3. Envie todos os arquivos e pastas para a raiz do repositório.
4. Confirme se existe exatamente este caminho:

```
assets/img/igreja-aquarela.jpg
assets/img/envelope-reference.jpg
index.html
```

5. Clique em `Commit changes`.
6. Aguarde 1 a 5 minutos.
7. Abra:

```
https://julioprecoma.github.io/convite-casamento-thaisegiovanni/
```

## Se ainda aparecer quebrado

Use uma janela anônima ou acrescente `?v=4` no final do link:

```
https://julioprecoma.github.io/convite-casamento-thaisegiovanni/?v=4
```
