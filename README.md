# Marco 0 — Home final

Versão funcional da Home conforme o modelo aprovado:

- Envelope inicial grande, com linhas finas e selo roxo.
- Clique/toque abre o convite suavemente.
- Igreja em aquarela como plano de fundo do menu.
- Menu em botões à direita, com ícones e seta.
- Rolagem habilitada apenas quando a tela do celular não comportar todos os botões.
- Layout responsivo para retrato e paisagem.

## Como publicar no GitHub Pages

1. Descompacte o ZIP.
2. Envie o conteúdo da pasta para a raiz do repositório.
3. Confirme que `index.html` está na raiz.
4. Abra a URL do GitHub Pages com cache busting, por exemplo:

`https://julioprecoma.github.io/convite-casamento-thaisegiovanni/?v=9`
