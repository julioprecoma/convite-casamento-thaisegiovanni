# Thaís & Giovanni — Home v7 Versão reconstruída do zero conforme o mockup aprovado. ## Como subir no GitHub Pages 1. Descompacte o ZIP. 2. Envie o conteúdo da pasta para a raiz do repositório. 3. O arquivo `index.html` precisa ficar na raiz. 4. Aguarde o GitHub Pages atualizar. 5. Abra com cache busting: `?v=7` no final do link. Exemplo: `https://julioprecoma.github.io/convite-casamento-thaisegiovanni/?v=7` ## O que editar ### Links dos botões No `index.html`, procure os links: - `href="convite.html"` - `href="cerimonia.html"` - `href="festa.html"` - `href="presentes.html"` - `href="hospedagem.html"` - `href="transporte.html"` - `href="presenca.html"` - `href="recados.html"` ### Imagem da igreja Arquivo usado: `assets/img/igreja-aquarela.jpg` Para trocar, substitua esse arquivo mantendo o mesmo nome. ### Posição do menu No CSS dentro do `index.html`, ajuste `.menu-list`: - `right:` move o menu para esquerda/direita. - `transform: translateY(-44%)` move verticalmente. - `width:` altera a largura dos botões. ### Tamanho do envelope No CSS, ajuste: - `.envelope-stage { width: ...; height: ... }` - `.envelope { height: ... }` ## Observação Esta versão não depende de arquivos CSS ou JS externos, para evitar falhas de carregamento no GitHub Pages.
